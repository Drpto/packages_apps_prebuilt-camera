# Inherit it from `device.mk` in device tree:
1. GCamGOPrebuilt
```
PRODUCT_PACKAGES += \
    GCamGOPrebuilt
```
2. Latest GCamGOPrebuilt packages (Experimental)
```
PRODUCT_PACKAGES += \
    GCamGOPrebuilt-V2
```
3. GrapheneOS Camera
```
PRODUCT_PACKAGES += \
    GrapheneCamera
```

# Clone this repository
```
git clone https://github.com/Drpto/packages_apps_prebuilt-camera.git packages/apps/prebuilt
```
